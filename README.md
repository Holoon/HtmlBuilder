# HtmlBuilder

![HTML](/HTML.png "LOGO")

Tool to build an HTML DOM in C# and export it to a string or to an IHtmlContent Enumerable. 

## Installation 

```
Install-Package Holoon.HtmlBuilder
```

Nuget package: https://www.nuget.org/packages/Holoon.HtmlBuilder/
